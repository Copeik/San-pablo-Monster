# Kit PixelLab · Plaza de la Farmacia

Sprites originales generados el 18-07-2026 mediante el MCP oficial de PixelLab.
Las fachadas usan cámara ortográfica `low top-down`: se ve el frente orientado
hacia el jugador y una porción de cubierta, como en un RPG portátil clásico.
El mobiliario usa `high top-down` para conservar una huella pequeña y legible.

Todos los PNG conservan fondo alfa, píxel nítido y no incluyen marcas ni
texto legible. Los lienzos se recortaron a sus píxeles visibles para que la
línea de fachada, la profundidad y las colisiones coincidan en el juego. Son
interpretaciones jugables, no reproducciones de negocios ni de rótulos reales.
