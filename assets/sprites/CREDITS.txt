hgss-npc-idle.png contains ten 32x32 idle frames cropped from that sheet. The
solid sheet background was converted to transparency; the sprite pixels were
otherwise left unchanged.

protagonist-walk.png / npc-guide-walk.png
========================================

Adapted from the two character references supplied by the project owner.
The 4×4 walking atlases were rebuilt with OpenAI image generation, chroma-key
extraction and local pixel-grid normalization. They contain four directions
(down, left, right, up) and the sequence neutral/left/neutral/right.

npcs/npc-01-* through npcs/npc-30-*
====================================

Original NPC roster created with OpenAI image generation using
protagonist-walk.png as the structural and pixel-art style reference. The flat
chroma backgrounds were removed locally and every sheet was normalized to the
same 4×4, 64-pixel-cell contract. Character prompts and final assets were made
for this fan project; no third-party sprite was copied into these 30 sheets.

doctor-potato-source.png / npcs/doctor-potato-walk.png
=====================================================

Character animation reference supplied by the project owner. The game reads
the permanent transparent 4×4 walking atlas directly. It was built
deterministically from the supplied 6×3 sheet, without generative changes to
the character, and follows the 64-pixel-cell contract used by overworld NPCs.
