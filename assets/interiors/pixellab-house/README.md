# Mobiliario de interiores PixelLab

Colección de 20 sprites PNG con fondo transparente, creada el 17 de julio de 2026 mediante el MCP oficial de PixelLab para `pokemon-adventure`. La cámara solicitada fue `top-down`, con alzado ortogonal frontal o lateral, píxel nítido y sin vistas diagonales.

Los archivos mantienen el lienzo original de 85 × 85 px para que compartan escala y encajen en la cuadrícula de 32 px del Modo Dios. Las alfombras se renderizan en la capa de suelo y no bloquean el paso; el resto incluye una huella de colisión ajustada a su base.
