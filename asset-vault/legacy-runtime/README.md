# Recursos legacy pendientes de corrección

Esta carpeta es la cuarentena lógica y no publicable. `nonstandard-assets-v0.json` enumera cada entidad, sus rutas actuales y los problemas exactos.

Los subdirectorios separan el trabajo por contrato. Los binarios que aún utiliza el juego conservan temporalmente su ruta original para no romper partidas ni mapas; el build estándar permanece cerrado hasta migrarlos.
