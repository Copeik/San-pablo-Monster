# Mundo no estándar

Aquí se registran los objetos de mundo activos que incumplen `world-asset-v1`. El perfil exige una carpeta estable por asset con `asset.png`, `manifest.json` y `credits.txt`, además de tamaño lógico, pivote, colisiones, etiquetas y SHA declarados. Las fuentes, previsualizaciones y ZIP no se consideran runtime y quedan fuera de todos los paquetes de publicación.

El inventario completo y la clasificación viven en `tools/assets/asset-inventory-v0.json`.
